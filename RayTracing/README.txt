RayTraceS_2Dlyr.m: MATLAB script that traces one Ss ray from the bottom of the model to the free surface
RayTracePmP_2Dlyr.m: MATLAB script that traces one Pmp ray from the free surface to the Moho and then back to the surface
AppRayTraceS_2Dlyr.m: MATLAB script that calls RayTraceS_2Dlyr.m to trace multiple Ss rays
AppRayTracePmP_2Dlyr.m: MATLAB script that calls RayTracePmP_2Dlyr.m to trace multiple Pmp rays
RayTrac_mod.txt: A sample input model file for the case with a 1200 km-wavelength LAB and flat CMB
